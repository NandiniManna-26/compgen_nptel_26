# Computational Genomics - M. tuberculosis Analysis
This repository contains practical bioinformatics work focused on the genomic analysis of *Mycobacterium tuberculosis* H37Rv.

## 🧬 Project Overview
As part of my computational genomics coursework, I performed sequence analysis and gene prediction on a genomic fragment (807 bp) of the H37Rv strain.

## 🛠️ Tools & Methods
- **EMBOSS Needle/Water:** Performed global and local sequence alignments.
- **EMBOSS getorf:** Predicted 8 potential Open Reading Frames (ORFs) on both forward and reverse strands.
- **EMBOSS pepstats:** Analyzed protein physical properties, including Isoelectric Point (pI) and Molecular Weight.
- **Linux/WSL:** Managed file systems and command-line biological tools.

## 📊 Key Findings
- **Fragment Length:** 807 bp.
- **ORF Count:** 8 potential protein-coding regions identified.
- **Biochemistry:** Analysis suggests a high GC-content typical of the *M. tuberculosis* genome, reflected in the amino acid composition.
